## ðŸŽ® **Get Started in 3 Steps**

### 1. **Install the Extension**
- Open Cursor
- Go to Extensions (Ctrl+Shift+X)
- Search for "Cost Katana AI Optimizer"
- Click Install

### 2. **Connect Your Account**
- Press `Ctrl+Shift+P` (or `Cmd+Shift+P` on Mac)
- Type "Cost Katana: Connect to Backend"
- Enter your API key or use magic link signup

### 3. **Start Saving Money**
- **Track a request**: Use "Cost Katana: Track AI Usage" after any AI interaction
- **Get recommendations**: Use "Cost Katana: Get Model Recommendations" before starting a task
- **Optimize prompts**: Select text and use "Cost Katana: Optimize Prompt"
